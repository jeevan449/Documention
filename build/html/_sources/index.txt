pyRestAuto's documentation!
===========================

PyRestAuto provides RestAPI automation framework comes with requests,json
parsing,logger,config and pyTest modules.

It is easy way write unittests for REST API's(POST,GET..) with and with out
authentication and includes logging and HTML reports.
